Shasta Documentation

If you are looking at this documentation on GitHub Pages (https://chanzuckerberg.github.io/shasta/), this documentation applies to the latest Shasta code on GitHub and might contain features that are not available in the Shasta version you are using. To obtain documentation that applies to a specific release, download the tar file for that release and, after extracting it, look at index.html in the docs directory.
